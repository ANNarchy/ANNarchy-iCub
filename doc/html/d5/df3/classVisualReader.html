Init Visual reader with given parameters for image resolution, field of view and eye selection.

Parameters
[in]eyecharacter representing the selected eye (l/L; r/R) or b/B for binocular mode (right and left eye image are stored in the same buffer)
[in]fov_widthoutput field of view width in degree [0, 60] (input fov width: 60째)
[in]fov_heightoutput field of view height in degree [0, 48] (input fov height: 48째)
[in]img_widthoutput image width in pixel (input width: 320px)
[in]img_heightoutput image height in pixel (input height: 240px)
[in]max_buffer_sizemaximum buffer length
[in]fast_filterflag to select the filter for image upscaling; True for a faster filter
Returns
True, if the initializatiion was successful. False if an error occured. Additionally, an error message is written to the error stream (cerr).
Typical errors:
  • arguments not valid: e.g. eye character not valid
  • YARP-Server not running
